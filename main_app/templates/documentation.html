    Как настроить проверку?

    Для настройки перейдите в:
    Проверки / Добавить.

    - введите любой заголовк, для идентификации проверки
    - введите url, который нужно проверять
    - выберите соответствующий тип проверки, или "простую проверку" для всех других сайтов

    Как настроить сложную проверку?

    Существует несколько видов сложных проверок:

    Проверка заполнения блока
    Определяет наличие элементов в выбранном блоке.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    Поле «Дополнительный параметр» оставлять пустым.

    Количество элементов превышает указанное
    Сравнивает количество элементов в блоке с установленным числом.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    В поле «Дополнительный параметр» введите число для сравнения.

    Количество элементов ниже указанного
    Сравнивает количество элементов в блоке с определенным числом.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    В поле «Дополнительный параметр» введите число для сравнения.

    Соответствие содержанию
    Проверяет совпадение контента блока с заданным текстом.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    В поле «Дополнительный параметр» введите искомый текст.

    Начало контента
    Проверяет, начинается ли контент блока с заданного фрагмента.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    В поле «Дополнительный параметр» введите искомый фрагмент.

    Окончание контента
    Проверяет, оканчивается ли контент блока заданным фрагментом.
    В поле «Проверяемый блок» укажите селектор с именем тега, например: div.my_class или span#my_id.
    В поле «Дополнительный параметр» введите искомый фрагмент.

    Регулярные выражения
    Проверяет контент по указанному регулярному выражению.
    В поле «Проверяемый блок» введите регулярное выражение для поиска контента.
    В поле «Дополнительный параметр» введите json, если данные в формате JSON, либо оставьте поле пустым.
    JSON содержит
    Проверяет JSON контент.
    В поле «Проверяемый блок» введите ключ.
    В поле «Дополнительный параметр» значение = 100.

    Управление уже настроенными проверками.

    Управлять и изменять проверки, можно перейдя в:
    Проверки / Список.

    В правом столбике находятся иконки управления:
    - Просмотреть краткую информацию по сложной проверке
    - Редактировать проверку
    - Запустить проверку
    - Активировать/Деактивировать проверку
    - Удалить проверку

    Как запустить проверку?

    Для одноразового запуска воспользуйтесь кнопкой на главной странице.
    или на странице Список
    Для регулярного выполнения проверок, установите периодические задания, задав необходимое время и подтвердив операцию нажатием кнопки "добавить".

    Как настроить уведомления в telegram?

    Чтобы настроить уведомления в Telegram, выполните следующие шаги:
    1. Перейдите в настройки.
    2. Найдите блок "Уведомления Telegram".
    3. Нажмите на ссылку, ведущую на бота Telegram.
    4. Откроется окно чата с ботом. Отправьте ему команду /start.
    5. Скопируйте ваш уникальный ID, который отправит вам бот.
    6. Вернувшись в настройки, вставьте этот ID в соответствующую форму для добавления нового пользователя.
    Альтернативный способ:
    1. Перейдите в настройки.
    2. Найдите блок "Уведомления Telegram".
    3. Скопируйте код подтверждения на странице настроек.
    4. Нажмите на ссылку, ведущую на бота Telegram.
    5. Откроется окно чата с ботом. Отправьте ему команду /start.
    6. Используйте специальную команду в боте формата /con [код подтверждения], заменяя [код подтверждения] вашим уникальным значением.
    Теперь уведомления будут приходить вам прямо в Telegram.
